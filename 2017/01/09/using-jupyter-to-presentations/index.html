Using jupyter to generate Presentations on reveal.js

Posted on Jan 09, 2017 By Ney Lemke

In this post I will describe the configuration and instalation of a system to produce slides using reveal.js. The great advantage of using this system is that you can produce interactive presentations that could be used as teaching tools for compouter science, physics, etc classes.

What you need to have installed:

  • conda
  • jupyter or jupyter lab

The first step is to install RISE

conda install --yes -c damianavila82 rise

This will install rise and all the tools need to use reveal.js on jupyter. The next step is to enable RISE nbextension, that means a small interface adjustment to jupyter so you can generate live presentaions.

jupyter nbextension install rise --py --sys-prefix
jupyter nbextension enable rise --py --sys-prefix

Start jupyter

jupyter notebook

You will see a new icon on jupyter interface.

To start using the system just go to Cell Toolbar -> Slide.

A very useful trick is to start your presentation notebook using:

from notebook.services.config import ConfigManager
from IPython.paths import locate_profile
cm = ConfigManager(profile_dir=locate_profile(get_ipython().profile))
cm.update('livereveal', {
              'theme': 'beige',
              'transition': 'zoom',
              'start_slideshow_at': 'selected',
})

So you can change reveal.js configuration. You can try the system by downloading this. You have to run all cells so the last command works.
