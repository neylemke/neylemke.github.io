## Environment

We will that assume the user has already installed these tools:

  • emacs
  • spacemacs http://https://github.com/syl20bnr/spacemacs
  • orgmode http://http://orgmode.org
  • latex
  • anaconda https://www.continuum.io/downloads
  • jupyter http://http://jupyter.org

## Python We will need to install some python packages to start

conda install jupyter_console 
conda install jupyter_client
conda install tornado
conda update ipython

## Emacs We will also need to install some packages on emacs: ob-python https://github.com/gregsexton/ob-ipython and ox-reveal https://github.com/yjwen/org-reveal. I have edit .spacemacs. I included a line on user-config section

(eval-after-load 'org '(load-library 'ox-reveal ))

I also included some languages to use on babel

(org-babel-do-load-languages
 'org-babel-load-languages
 '((python . t)
   (latex . t)
   (shell . t)
   (ipython . t)
   (elisp . t)
   ))
)

I also edited aditional packages.

dotspacemacs-additional-packages '(
                                   ecb
                                   ob-ipython
                                   ox-reveal
                                   )
